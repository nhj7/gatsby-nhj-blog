맥에서 nginx를 설치하고 설정을 해보도록 하자. 

  

우선 brew install nginx 로 설치해주자. 그 다음 명령어

  

시작 : nginx 

중지 : nginx -s stop

재시작 : nginx -s reload

  

환경 설정 : vi /usr/local/etc/nginx/nginx.conf
